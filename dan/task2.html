4 Components of the Computer System

  • Hardware
  • Software
  • Dataware
  • Peopleware/Liveware

Hardware

Hardware is any physical component of a computer system that you can see and touch.


Examples:

  • Mouse
  • Keyboard
  • Monitor
  • RAM
  • GPU

Software

The set of instructions, programs, and data that tell a computer what to do and how to do it, performing specific tasks and functions.

Examples:

  1. Visual Studio Code
  2. Paint 3D
  3. XAMPP
  4. Termux
  5. Neovim

Dataware

A central repository for storing and analyzing structured and semi-structured data from multiple sources, used for business intelligence and reporting

Examples:

  1. Amazon Redshift
  2. Microsoft Azure Synapse Analytics
  3. Snowflake

Peopleware/Liveware

Peopleware refers to the human element of computer systems, including the people, skills, and processes involved in developing, using, and managing hardware and software.

Examples:

  1. Software Engineers
  2. Game Developers
  3. Web Developers
